# Nutrify

> **Nutrify** - это Ruby-гем для комплексного анализа состава продуктов питания и БАДов. Он помогает определить безопасность продукта с учётом индивидуальных особенностей: аллергий, диетических ограничений и состояния здоровья.

## Возможности:
1. **Анализ Е-добавок и ингредиентов**
   - распознавание Е-кодов в любом формате (Е100, e100, E-100, E 100)
   - поиск по названиям и синонимам (глутамат натрия, monosodium glutamate, усилитель вкуса, E621)
   - детальная информация о каждой добавке:
      * назначение и происхождение (животное/растительное/синтетическое)
      * уровень опасности и потенциальные риски
      * допустимая суточная норма
      * разрешена ли в разных странах
2. **Персонализированный анализ**
   - учёт индивидуальных особенностей:
      * аллергии (лактоза, глютен, орехи и др.)
      * тип диеты (веган, вегетарианец, безглютеновая)
      * возрастные ограничения
   - персональная оценка: "Этот продукт вам НЕ подходит, потому что..."
3. **Умные функции**
   - парсинг состава — извлечение всех ингредиентов из текстовой строки
   - оценка безопасности — числовой рейтинг продукта от 0 до 100
   - анализ комбинаций — выявление опасных сочетаний добавок
   - сравнение продуктов — какой состав безопаснее
   - генерация отчётов — красивый HTML или читаемый консольный вывод
4. **Интеграции**
   - подключение к Open Food Facts API для получения актуальных данных
